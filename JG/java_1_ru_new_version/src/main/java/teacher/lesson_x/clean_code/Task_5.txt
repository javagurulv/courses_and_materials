Цель тренировки: разбиения кода на методы (продолжение).

Дополнительное ограничение: длина каждого метода не больше 3 строк кода!

Да, в реальной жизни это ограничение кажется просто абсурдным. Но
помните, мы не пишем сейчас код реального проекта, мы проводим
целенаправленную тренировку. Наша цель: научиться разбивать код
программы на маленькие методы.

После выполнения таких экстремальных тренировок при решении реальной
задачи вы сами будете стараться делить код на не большие (пускай
и не 3 строки) методы, тем самым улучшая структуру и читабельность кода.

Задача 1.3: скопируйте ваше первое решение в новый пакет
и там проведите рефакторинг кода так, чтобы он удовлетворял поставленному
ограничению (длина каждого метода не больше 3 строк кода).

Задача 1.4: в новом пакете решите задачу с нуля учитывая дополнительное
ограничение (длина каждого метода не больше 3 строк кода).
