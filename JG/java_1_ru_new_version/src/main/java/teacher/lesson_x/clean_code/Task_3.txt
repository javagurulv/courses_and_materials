Первое решение выбранной задачи.

После того как вы выбрали задачу, просто решите её как умеете. При
первом решении вы вникните в бизнес, проясните все непонятные моменты.
Это позволит вам при повторных решениях концентрироваться на
технических аспектах, а не на бизнес проблемах. Напомню, что
наша цель это улучшение качества нашего кода, а это сугубо
технический аспект.
