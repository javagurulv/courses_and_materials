Как можно работать над улучшением качества своего кода?

Один из способов это тренировка/практика нацеленная на оттачивание
навыков написания чистого кода (clean code). Понятие чистого кода
(clean code) включает в себя большое количество эмпирических правил
(утверждений), следуя которым можно сделать свой код лучше и качественнее.

Как организовать и проводить такую тренировку?

Во-первых, нужно выбрать не сложную бизнес задачу. Да, именно не
сложную! Потому что для нас целью тренировки будет не решение
сложных бизнес задач, а оттачивание технических навыков написания
кода. Во время тренировки мы должны будем концентрироваться на
технических навыках, а не на сложно бизнес проблеме. Задача
расчёта стоимости страхового полюса и игра Game of Life идеально
подходят для этой цели.

Очень важно, чтобы решение задачи не требовало внешних интеграций
таких как базы данных, веб-сервисов и тяжёлых фреймворков. Наша
цель не состоит в том, чтобы научиться создавать такие интеграции
или применять фреймворки (это может быть отдельной темой для
практики), наша задача улучшить свои навыки в написании кода.
