Для оттачивания (тренировки) конкретного технического навыка
нужно при решении задачи концентрироваться именно на нём.
Один из вариантов как это можно делать заключается в накладывании
дополнительных ограничений на техническое решение.

Цель: тренировка разбиения кода на методы.

Дополнительное ограничение: длина каждого метода не больше 5 строк кода!

Задача 1.1: скопируйте ваше первое решение в новый пакет (version 2)
и там проведите рефакторинг кода так, чтобы он удовлетворял поставленному
ограничению (длина каждого метода не больше 5 строк кода).

Задача 1.2: в новом пакете решите задачу с нуля учитывая дополнительное
ограничение (длина каждого метода не больше 5 строк кода).
