Цель тренировки: научиться использовать функциональное
                 программирование (Stream API).

Начиная с Java 8 в синтаксис языка программирования добавили такие
вещи как Optional, лямбда выражения и Stream API. Эти вещи открыли
возможность использования основ функционального программирования.
С тех пор современный код невозможно себе представить без этих
вещей.

Функциональный стиль работы с коллекциями упрощает код, убирает из
него повторяющиеся громоздкие конструкции циклов и даёт возможность
программисту концентрировать своё внимание на декларативном объявлении
функций, а не на скучной их реализации.

Задача 1.17: возьмите одно из своих решений и перепишите его с
использованием Stream API, лямбда выражений и Optional там где
это возможно.

Задача 1.18: решите задачу с чистого листа сразу используя
функциональное программирование.
