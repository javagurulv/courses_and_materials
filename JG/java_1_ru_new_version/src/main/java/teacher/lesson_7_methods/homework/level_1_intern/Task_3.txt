При создании метода приходится ответить на три вопроса:
1. как будет называться метод?
2. какие параметры нужно подать методу на вход?
3. что будет возвращать метод?

Для метода из класса FizzBuzz, который определяет делиться ли
число на 3 без остатка или нет:
1. isDividedBy3
2. само число, то есть int number
3. true или false - делится ли число на 3 без остатка или нет - это
   тип данных boolean

После этого мы можем легко заполнить шаблон нового метода:

returnType methodName(parameters) {

}

boolean isDividedBy3(int number) {

}

Осталось только написать тело метода!
