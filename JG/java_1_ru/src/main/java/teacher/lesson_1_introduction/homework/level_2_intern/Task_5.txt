Запросить у преподавателя и получить доступ к
GitHub репозиторию курса.

1. Преподавателю нужно отослать свой логин от GitHub.
2. Преподаватель добавит вас в пользователи проекта.
3. Вам на е-мейл от GitHub придёт письмо с проcьбои подтвердить
   своё участие в проекте.
4. Подтвердите своё участие и у вас появится доступ с правами на запись.

После этого вы сможете добавлять свой код в репозиторий!
